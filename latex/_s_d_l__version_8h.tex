Macro to determine S\+DL version program was compiled against. 

This macro fills in a \mbox{\hyperlink{struct_s_d_l__version}{S\+D\+L\+\_\+version}} structure with the version of the library you compiled against. This is determined by what header the compiler uses. Note that if you dynamically linked the library, you might have a slightly newer or older version at runtime. That version can be determined with \mbox{\hyperlink{_s_d_l__version_8h_a4204ecbb600f51ee50515979e60bf8f3}{S\+D\+L\+\_\+\+Get\+Version()}}, which, unlike \mbox{\hyperlink{_s_d_l__version_8h_a7154bc5c2a95a644b5187e4225cdcbd0}{S\+D\+L\+\_\+\+V\+E\+R\+S\+I\+O\+N()}}, is not a macro.

Represents the library\textquotesingle{}s version as three levels\+: major revision (increments with massive changes, additions, and enhancements), minor revision (increments with backwards-\/compatible changes to the major revision), and patchlevel (increments with fixes to the minor revision).

Get the version of S\+DL that is linked against your program. 

If you are linking to S\+DL dynamically, then it is possible that the current version will be different than the version you compiled against. This function returns the current version, while \mbox{\hyperlink{_s_d_l__version_8h_a7154bc5c2a95a644b5187e4225cdcbd0}{S\+D\+L\+\_\+\+V\+E\+R\+S\+I\+O\+N()}} is a macro that tells you what version you compiled with.
