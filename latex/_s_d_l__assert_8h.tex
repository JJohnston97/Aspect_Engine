Set an application-\/defined assertion handler. 

This allows an app to show its own assertion UI and/or force the response to an assertion failure. If the app doesn\textquotesingle{}t provide this, S\+DL will try to do the right thing, popping up a system-\/specific G\+UI dialog, and probably minimizing any fullscreen windows.

This callback may fire from any thread, but it runs wrapped in a mutex, so it will only fire from one thread at a time.

Setting the callback to N\+U\+LL restores S\+DL\textquotesingle{}s original internal handler.

This callback is N\+OT reset to S\+DL\textquotesingle{}s internal handler upon \mbox{\hyperlink{_s_d_l_8h_afdabaf714781099083592051f2d9ac11}{S\+D\+L\+\_\+\+Quit()}}!

Return S\+D\+L\+\_\+\+Assert\+State value of how to handle the assertion failure.
